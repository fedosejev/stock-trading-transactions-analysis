## Install

1. Create `config.json` file by copying `config.example.json`.
2. Edit `config.example.json` as needed.
3. `npm install --global http-server`.
4. `npm install`.
5. `cd build && http-server`.
6. Open `http://localhost:8080` in Google Chrome.
